1. 노드 찾기

    - $('찾을노드')

    - 노드의 컬렉션(개체와 비슷)을 반환(return)

    - '찾을노드' : css의 Selector 와 같은 형태로 사용할 수 있음

  2. 자식 노드 찾기

    2-1. CSS Selector 형태 사용하기

      - CSS Selector 를 사용하는 것과 동일하게 사용할 수 있음.

    2-2. children() 함수 사용하기

      - 직계 자식들을 찾아주는 함수

      - 직계 자식들 중 특정 상태의 자식을 찾을 수 있음.

    2-3. find() 함수 사용하기

      - 직계 자식 뿐만 아니라 자손들도 찾아주는 함수

      - 넣어주는 인수에 따라서 자손 노드 찾는 방법이 다양해 짐
