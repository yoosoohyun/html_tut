# Tutorial

## 개념

1. HTML (Hyper Text Markup Language)
    - 웹 페이지의 논리 구조를 통해 콘텐츠를 표시하는 언어
    - HTML5 추가된 요소 : header, footer, section, nav, article, aside

2. CSS
    2-1. CSS2
        - 개체(대상) 스타일 관련 CSS
        - 레이아웃(박스모델) 관련 CSS

    2-2. CSS3
        - 개체(대상) 스타일 관련 CSS
        - 변환, 애니메이션 관련 CSS
        - flex : 레이아웃 CSS

## 웹폰트
1. 온라인 서비스
    - 구글 폰트
    - 유료 웹폰트 서비스 ( 타이포링크 )

2. 로컬 파일
    - eot, woff 파일 변환
    - @font-face 직접 정의한 후에 사용
